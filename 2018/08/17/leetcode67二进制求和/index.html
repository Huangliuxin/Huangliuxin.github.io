leetcode67二进制求和

发表于 2018-08-17 | 分类于 leetcode

题目

给定两个二进制字符串，返回他们的和（用二进制表示）。

输入为非空字符串且只包含数字 1 和 0。

示例2

输入: a = “1010”, b = “1011”

题解
